Backgrounds

Background Image

A propriedade background-image é usada diretamente no CSS, adiciona uma imagem no projeto.

Bachground Repeat

A propriedade background-repeat é usada para definir se a imagem irá repetir ou não e se sim, para qual direção ela irá repetir.

Bachgroud Size Cover

A propriedade background-cover é usada para definir que a imagem cobrirar todo o conteúdo, independênte do tamanho desse conteúdo. A imagem pode ficar estourada.

Background Attachment Fixed

A propriedade background-attachment: fixed é usada para que a imagem fique fixa na página, ao scrolar a página, nossa imagem ficará fixa no fundo da página, enquanto o outros conteúdos rolarão na tela.

Background Position

A propriedade background-position define qual a posição da imagem: direita, esquerda ou no centro horizontalmente e também pode ser: em cima, em baixo ou no centro verticalmente. Podemos usar vários tipos de unidade para definir a posição da imagem, como: pixel, porcentagem, etc.

Background: forma encurtada

Podemos usar todas as propriedades do background de forma encurtada. Indicando a propriedade background e inserindo os valores: background: green url(' ') no-repeat left top
